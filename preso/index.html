Steps to setup AJAX with WP


  1. WP: Tell JS where to find data
  2. WP: Create the data for JS
  3. JS: AJAX call to retrieve data
  4. JS: Do something with the data

Use case:

Passing data to JS after page load from WP.


Ideal:

When you need to access to new data from the server without effecting the state of an existing page.

wp_localize_script();

// Setup our data
$myDataArray = array(
	'ajax_url' => admin_url( 'admin-ajax.php' ),
);

// Pass data to myscript2.js on page load
wp_localize_script( "myScript2", "myLocalizedData2", $myDataArray );
Gist

AJAX actions in WP

/**
 * Setup JSON Ajax endpoint for Javascript async access to WP data 
 */

add_action('wp_ajax_my_action_1', 'my_callback_function_1');
add_action('wp_ajax_nopriv_my_action_1', 'my_callback_function_1');

function my_callback_function_1(){

	header('Content-Type: text/html');
	echo '<p>HTML from WP for Javascript to use</p>';
	die(); // Prevent WP -1

}

JS (jQuery) to get the data

// Access WP data using $.ajax()
$.ajax({
	method: "POST",
	dataType: "html",
	url: myLocalizedData2.ajax_url,
	data: {action:'my_action_1'},

}).done(function(myAjaxData){ // using promise

 	// Place Data in DOM

Let's load some posts

Steps

  1. WP: Tell JS where to point using wp_localize_script()
  2. WP: Create new Ajax action and return post data
  3. JS: AJAX call to retrieve data and add to page

WP: Tell JS where to point using wp_localize_script()


// Setup our data
$myDataArray = array(
	'ajax_url' => admin_url( 'admin-ajax.php' ),
);

// Pass data to myscript4.js on page load
wp_localize_script( "myScript4", "myLocalizedData3", $myDataArray );

WP: Create new Ajax action and return post data

add_action('wp_ajax_my_action_4', 'my_callback_function_4');
add_action('wp_ajax_nopriv_my_action_4', 'my_callback_function_4');

function my_callback_function_4(){

	// Get some data from WP
	$args = array(
		'post_type'=>'post',
		'posts_per_page' => 10
	);
	$posts_array = get_posts($args);

	// Return data as JSON string
	wp_send_json($posts_array);

}
Example

JS: AJAX call to retrieve data

$.ajax({
	method: "POST",
	dataType: "JSON",
	url: myLocalizedData3.ajax_url,
	data: {action:"my_action_4"},
}).done(function(myAjaxData){
	$html = "";

  	$.each(myAjaxData,function(index,value){
	$html +="<h3>"+value.post_title+"</h3><br>";
	$html +="<div>"+value.post_content+"</div><br><br>";
  	});
  	
  	$('#displayData').append($html);
});
