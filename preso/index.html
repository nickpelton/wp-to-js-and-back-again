AJAX
Asynchronous JavaScript and XML

Set of tools to asynchronously communicate with a server
in the background without reloading the page.

What happens during AJAX?
(simplified)

  1. Javascript in a browser sends a HTTP request to the server
  2. The server receives the HTTP request, processes it, and returns a response
  3. JS recieves the response back from the server then does something with it.

Normal page load

Rinse & Repeat

What happens during AJAX?

All in the background

Goals of Ajax?

  • Preserving the browser state
  • Reduced requests and or server usage
  • Speed
  • Increase user experience

AJAX Examples

  • Load more posts without reload (IE: Infinite Scroll)
  • Live search filter
  • Track statistics
  • Near Real Time data (live chat)
  • WordPress Media Library (3.0+)
  • Much more.

Passing data from WP-to-JS
Using Ajax

Use case:

Passing data to JS after page load from WP.


Ideal:

When you need to access to new data from the server without effecting the state of an existing page.

Two Worlds

The Browser and the Server are indepenant of each other. Neither knows the other exists unless you tell them where to go.

Just like an A tag link, we need to tell Javascript where it needs to look for new data.

wp_localize_script();


// Setup our data
$myDataArray = array(
	'ajax_url' => admin_url( 'admin-ajax.php' ),
);

// Pass data to myscript2.js on page load
wp_localize_script( "myScript2", "myLocalizedData2", $myDataArray );
Gist

admin-ajax.php


// Create an AJAX endpoint for JS to get data from
// 
// [site_url]/wp-admin/admin-ajax.php?action=action_name

// Add ajax actions. "nopriv" is for anonymous in users
add_action('wp_ajax_nopriv_action_name', 'callback_function'); 
add_action('wp_ajax_action_name', 'callback_function'); 

function callback_function(){

	// Do something
	die();

}

Let's load some posts

Bring it full circle

Steps

  1. Localize the Ajax URL for JS
  2. Create new Ajax action (endpoint) in WP
  3. Load the latest posts and return in JSON
  4. Add data to the page using JS

Loclalize the Ajax URL for JS


// Setup our data
$myDataArray = array(
	'ajax_url' => admin_url( 'admin-ajax.php' ),
);

// Pass data to myscript4.js on page load
wp_localize_script( "myScript4", "myLocalizedData3", $myDataArray );

Create new Ajax action (endpoint) in WP


add_action('wp_ajax_my_action_4', 'my_callback_function_4');
add_action('wp_ajax_nopriv_my_action_4', 'my_callback_function_4');

function my_callback_function_4(){

	// Do something
	die();

}

Load latest posts and return in JSON

add_action('wp_ajax_my_action_4', 'my_callback_function_4');
add_action('wp_ajax_nopriv_my_action_4', 'my_callback_function_4');

function my_callback_function_4(){

	// Get some data from WP
	$args = array(
		'post_type'=>'post',
		'posts_per_page' => 10
	);
	$posts_array = get_posts($args);

	// Return data as JSON string
	wp_send_json($posts_array);

}
Example

Add data to the page using JS

$.ajax({
	method: "POST",
	dataType: "JSON",
	url: myLocalizedData3.ajax_url,
	data: {action:"my_action_4"},
}).done(function(myAjaxData){
	$html = "";

  	$.each(myAjaxData,function(index,value){
	$html +="<h3>"+value.post_title+"</h3><br>";
	$html +="<div>"+value.post_content+"</div><br><br>";
  	});
  	
  	$('#displayData').append($html);
});
